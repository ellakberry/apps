
 openexr (HDR Image File Format & Library) 
 
 OpenEXR is an image file format and library developed by Industrial 
 Light & Magic, and later released to the public. It provides support 
 for high dynamic range and a 16-bit floating point "half" data type 
 which is compatible with the half data type in the Cg programming 
 language. 
 
 Homepage: http://www.openexr.org/
