
 xfig (interactive drawing tool) 
 
 This package contains the binary for xfig, an interactive drawing tool 
 under X. Xfig can create fig, epsf, and LaTeX files with various 
 options. You also have to install the transfig package (transfig.tgz) 
 so that xfig can export fig files as epsf and LaTeX files. This 
 package also contains a set of basic examples of fig files.
