
 herqq (UPnP library) 
 
 Herqq UPnP (HUPnP) is a software library for building UPnP devices 
 and control points conforming to the UPnP Device Architecture 
 version 1.1. It is designed to be simple to use and robust in 
 operation. It is built using C++ and the Qt Framework. 
 
 Homepage: http://www.herqq.org/
