 lrzip (Long Range ZIP) l
 l
 LRZIP is a file compression program designed to do particularly well l
 on very large files containing long distance redundancy. The larger l
 the file and the more memory you have, the better the compression l
 advantage this will provide. A variety of compression options allow l
 optimizing for size or speed. l

 rzip (a large-file compression program)

 on very large files containing long distance redundancy.
