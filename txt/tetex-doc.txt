
 tetex-doc (Documentation for teTeX) 
 
 This package contains documentation for AMSTeX, BibTeX, extended plain 
 format, font packages, format files (babel, texdraw, pstricks, xypic), 
 LaTeX, MetaPost, Makeindex, pdftex, the Kpathsea library, as well as 
 general information about TeX such as a list of ftp servers, a draft 
 for the TeX Directory Standard (TDS), The TeX and LaTeX Catalogue, 
 TeX FAQs, and more.
